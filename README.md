# 🧮 Smart Calculator

A **premium web-based calculator** with a cinematic dark blue theme, glassmorphism design, and smooth animations — built as part of my web development journey.

---

## ✨ Features

- 🎬 **Cinematic Title Page** — animated grid background, floating particles & math symbols
- 💎 **Glassmorphism UI** — modern glass effect with shine highlights
- 🔢 **Full Calculator** — addition, subtraction, multiplication, division
- ➕ **Extra Functions** — percentage (%), toggle sign (+/−), decimal support
- 💡 **Neon Display** — Orbitron font with glowing cyan text
- ⌨️ **Keyboard Support** — type numbers & operators directly
- 📱 **Responsive Design** — works on mobile & desktop

## 📂 How to Use

1. Clone or download this repository
2. Open `index.html` in any browser
3. Click **Launch Calculator** on the title page
4. Start calculating! 🎉

```bash
git clone https://github.com/memonaabdulrazzaq09-oss/calculator.git
cd calculator
open index.html
```

---

## ⌨️ Keyboard Shortcuts

| Key | Action |
|-----|--------|
| `0–9` | Enter numbers |
| `+ - * /` | Operators |
| `Enter` or `=` | Calculate |
| `Escape` | Clear (AC) |
| `Backspace` | Delete last digit |
